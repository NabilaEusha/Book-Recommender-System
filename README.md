# Book Recommender System

A simple book recommendation web app built with Flask and collaborative filtering.

## Features

- **Top 50 Books**: Browse popular books based on ratings and votes
- **Smart Recommendations**: Get personalized book suggestions using cosine similarity
- **Clean UI**: Modern, responsive design

## Tech Stack

- Flask
- Pandas & NumPy
- Scikit-learn (Cosine Similarity)
- Bootstrap 5

## Installation

```bash
# Clone the repo
git clone https://github.com/NabilaEusha/Book-Recommender-System.git
cd book_recommender_system

# Install dependencies
pip install -r requirements.txt

# Run locally
python app.py
```

Visit `http://localhost:5000`

## Dataset

Uses the Book-Crossing dataset with:
- 271k books
- 1.1M ratings
- 278k users
